一、简介

  • Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯；
  • 它允许人们使用易读易写的纯文本格式编写文档，Markdown 文件可以轻松地转换为其他格式，如 HTML、PDF、Word 等。这样，在需要发布到不同平台或媒体时，你只需将 Markdown 文件转换为目标格式即可。
  • 由于Markdown的轻量化、易读易写特性，并且对于图片，图表、数学式都有支持，目前许多网站都广泛使用Markdown来撰写帮助文档或是用于论坛上发表消息，如Github、CSDN、简书等。

二、语法

1 常用语法

markdown中比较常用的几个符号：

  1. #：标题
  2. *：粗体、斜体
  3. >：引用块
  4. $：latex数学公式
  5. `：代码


2 一些示例

代码示例：

#include 

using namespace std;

int main(){
  cout << "hello world!" << endl;

  return 0;
}

公式示例：

如果过集合 $C$ 中任意两点的直线都在 $C$ 内, 则称 $C$ 为仿射集, 即
$x_1, x_2 \in C \Longrightarrow \theta x_1+(1-\theta) x_2 \in C, \forall \theta \in \mathbb{R} .$
